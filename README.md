# 🏛️ موقع مدينة السلط الأردنية

موقع تراثي فاخر عن مدينة السلط — جوهرة الأردن وتراث اليونسكو 2021

---

## 🚀 طريقة التشغيل

### 1. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 2. تشغيل الموقع
```bash
python app.py
```

### 3. افتح المتصفح على
```
http://localhost:5000
```

---

## 📁 هيكل المشروع

```
salt-city/
├── app.py              ← سيرفر Flask (Python)

├── templates/
│   └── index.html      ← صفحة الموقع الرئيسية

```

---

## ✨ مميزات الموقع

- **Hero Section** — خلفية متحركة مع إحصائيات المدينة
- **قسم عن المدينة** — تاريخ السلط وطابعها المعماري
- **المعالم التاريخية** — 6 معالم مع فلتر حسب الفئة ونافذة تفاصيل
- **معرض الصور** — شبكة صور بتأثيرات hover احترافية
- **شريط اليونسكو** — تسليط الضوء على التصنيف العالمي 2021
- **دليل الزائر** — معلومات الوصول والسياحة
- **تصميم RTL كامل** — عربي أصيل

---

## 🎨 تقنيات الموقع

- **Backend:** Python Flask
- **Frontend:** HTML5 + CSS3 + Vanilla JS
- **Fonts:** Amiri (عربي كلاسيكي) + Cairo (حديث)
- **API Routes:**
  - `GET /api/landmarks` — جميع المعالم
  - `GET /api/landmarks?category=تراث` — فلتر حسب الفئة
  - `GET /api/landmark/<id>` — تفاصيل معلم واحد
